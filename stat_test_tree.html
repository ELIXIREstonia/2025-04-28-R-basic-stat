Statistical Test Decision Tree

Important Considerations Before Starting:

  1. Goal: What question are you trying to answer?
  2. Data Type:
    • Continuous (Interval/Ratio): Measurable, ordered, equal intervals (e.g., height, temp).
    • Ordinal: Ordered categories, unequal intervals (e.g., Likert scale, rank).
    • Nominal (Categorical): Unordered categories (e.g., gender, color).
  3. Number of Variables: How many variables are involved?
  4. Independent vs. Paired Samples: Are groups separate individuals or related measurements (e.g., before/after)?
  5. Assumptions: Check requirements for parametric tests (Normality, Homogeneity of Variances, Independence). Use non-parametric tests if violated.

START HERE: What is your primary goal?

Goal 1: Compare Groups (Means/Medians/Distributions)

  • How many groups are you comparing?
    • One Group (Compare sample to a known value):
      • Variable Data Type?
        • Continuous:
          • Assumptions met (Normality)?
            • Yes: One-Sample t-test
            • No: One-Sample Wilcoxon Signed-Rank Test
        • Nominal (Observed vs. Expected Frequency): Chi-Square Goodness-of-Fit Test
    • Two Groups:
      • Independent or Paired Samples?
        • Independent Groups:
          • Outcome Variable Data Type?
            • Continuous:
              • Assumptions met (Normality, Equal Variances)?
                • Yes: Independent Samples t-test
                • No (Normality violated): Mann-Whitney U Test (Wilcoxon Rank-Sum)
                • No (Equal Variances violated): Welch's t-test
            • Ordinal: Mann-Whitney U Test
            • Nominal (Compare Proportions/Frequencies): Chi-Square Test of Independence or Fisher's Exact Test (small samples) / Z-test for two proportions
        • Paired Groups:
          • Outcome Variable Data Type?
            • Continuous:
              • Assumptions met (Normality of differences)?
                • Yes: Paired Samples t-test
                • No: Wilcoxon Signed-Rank Test
            • Ordinal: Wilcoxon Signed-Rank Test
            • Nominal (e.g., change Yes/No): McNemar's Test
    • More Than Two Groups (>2):
      • Independent or Related (Repeated Measures)?
        • Independent Groups:
          • Outcome Variable Data Type?
            • Continuous:
              • Assumptions met (Normality, Equal Variances)?
                • Yes: One-Way ANOVA (+ Post-hoc tests e.g., Tukey)
                • No: Kruskal-Wallis H Test (+ Post-hoc tests e.g., Dunn’s)
            • Ordinal: Kruskal-Wallis H Test
            • Nominal (Frequencies across groups): Chi-Square Test of Independence
        • Related Groups (Repeated Measures):
          • Outcome Variable Data Type?
            • Continuous:
              • Assumptions met (Normality, Sphericity)?
                • Yes: Repeated Measures ANOVA
                • No (Sphericity violated): Repeated Measures ANOVA with correction (e.g., Greenhouse-Geisser)
                • No (Normality violated): Friedman Test
            • Ordinal: Friedman Test
            • Nominal: Cochran's Q Test

Goal 2: Examine Relationship/Association Between Two Variables

  • What are the data types of the TWO variables?
    • Both Continuous:
      • Linear relationship? Assumptions met?
        • Yes: Pearson Correlation (r)
        • No: Spearman Rank Correlation (rho) or Kendall's Tau
    • Both Ordinal: Spearman Rank Correlation (rho) or Kendall's Tau
    • Both Nominal: Chi-Square Test of Independence or Fisher's Exact Test (small samples)
    • One Continuous, One Nominal (Dichotomous - 2 categories): Independent Samples t-test / Mann-Whitney U / Point Biserial Correlation
    • One Continuous, One Nominal (>2 categories): Relates to ANOVA framework.
    • One Continuous, One Ordinal: Spearman Rank Correlation (rho) (often used)

Goal 3: Predict an Outcome (Regression Analysis - Brief Overview)

  • What is the data type of the Outcome (Dependent) Variable?
    • Continuous: Linear Regression (Simple or Multiple)
    • Nominal (Dichotomous - 2 categories): Binary Logistic Regression
    • Nominal (>2 categories): Multinomial Logistic Regression
    • Ordinal: Ordinal Logistic Regression

Disclaimer:

This tree covers common scenarios but isn’t exhaustive. Test choice depends on the specific research question, study design, data distribution, and sample size. Always verify test assumptions. Consulting statistical resources or a statistician is recommended for complex cases.
